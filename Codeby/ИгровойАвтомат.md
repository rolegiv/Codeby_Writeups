# Шифрование значения hash для запроса

```
import hashlib

result='777'

hash=hashlib.md5(hashlib.sha256(hashlib.sha1(hashlib.sha256(hashlib.sha1(result.encode('utf-8')).digest()).digest()).digest()).digest()).hexdigest()

print(hash)
```
